# C++ Fundamentals

Repositorio con ejercicios realizados durante el curso de **Principios de Programación** utilizando C++.

El objetivo de este repositorio es documentar mi proceso de aprendizaje, comenzando desde conceptos básicos hasta ejercicios más complejos.

## Contenido

* 1- Elementos básicos de C++
* 2- Estructuras de Control
* 3- Getchar()/Putchar()
* 4- Arreglos
* 5- Funciones

Los ejercicios están organizados por tema y numerados aproximadamente desde los más simples hasta los más complejos.
